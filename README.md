# Student Management System (JDC Portal)

An industrial-grade Student Management System built using a modern full-stack architecture. This portal allows seamless tracking and management of courses, enrollments, and student-teacher interactions with robust role-based access control (Admin, Teacher, Student).

---

## 🚀 Tech Stack

### Frontend
*   **Framework:** React (Vite) with TypeScript
*   **State Management:** Zustand
*   **Form Handling:** React Hook Form
*   **Styling & UI:** Tailwind CSS v4 & Shadcn UI
*   **HTTP Client:** Axios (with centralized Interceptors)

### Backend
*   **Framework:** Spring Boot 3.x
*   **Security:** Spring Security (OAuth2 & JWT Ready / Current: Open API for development)
*   **Database:** MySQL
